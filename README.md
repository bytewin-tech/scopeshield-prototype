# ScopeShield Prototype

Frontend-only micro SaaS prototype for **ScopeShield** — an AI-assisted proposal/SOW red-flag checker for freelancers, consultants, and small agencies.

## What’s included

- Polished landing page hero and positioning
- Interactive proposal input / analyze screen
- Results dashboard with risk score, flagged phrases, missing clauses, and suggested rewrites
- Template library and export actions
- Seeded sample proposals for realistic demo behavior

## Stack

- Static HTML
- CSS with embedded design tokens via variables
- Vanilla JavaScript for sample loading, analysis rendering, copy/export actions

## Local usage

Open `index.html` directly in a browser, or serve the directory with any static server.

## Deployment notes

This prototype is suitable for static hosting on Netlify.

Primary files:
- `index.html`
- `styles.css`
- `app.js`

## Product positioning

ScopeShield is positioned as a fast pre-send margin protection tool rather than a full legal platform.
